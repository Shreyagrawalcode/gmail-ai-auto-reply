# Gmail AI Auto-Reply Automation

## What it does
Automatically reads incoming Gmail, filters promotions,
analyzes real emails with AI, and sends replies hands free.

## How it works
→ Detects when a new email arrives
→ Checks if it's promotional (and skips it)
→ If it's a real email, AI reads and analyzes it
→ Decides if a reply is needed
→ Sends the reply automatically

## Built with
n8n · Gmail API · OpenAI

## Setup
1. Import the .json into your n8n instance
2. Add your Gmail and OpenAI credentials
3. Activate the workflow
